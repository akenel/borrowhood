{% if lang == 'it' %}Estrazione Verificabile{% else %}Provably Fair Draw{% endif %}

{% if lang == 'it' %}Quando l'organizzatore clicca "Estrai vincitore":{% else %}When the organizer clicks "Draw Winner":{% endif %}

  1. {% if lang == 'it' %}Il sistema genera un codice segreto casuale (seed){% else %}The system generates a random secret code (seed){% endif %}
  2. {% if lang == 'it' %}Elenca tutti i biglietti confermati in ordine{% else %}Lists all confirmed tickets in order{% endif %}
  3. {% if lang == 'it' %}Combina seed + lista e calcola SHA-256 (una funzione matematica){% else %}Combines seed + list and runs SHA-256 (a math function){% endif %}
  4. {% if lang == 'it' %}Il risultato determina il biglietto vincente{% else %}The result determines the winning ticket{% endif %}

{% if lang == 'it' %}Perché conta:{% else %}Why it matters:{% endif %}

{% if lang == 'it' %}Dopo l'estrazione, il seed viene pubblicato. Chiunque può verificare che il vincitore è stato scelto in modo equo.
