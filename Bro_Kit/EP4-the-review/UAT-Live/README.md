# UAT-Live -- EP4 Monkey Testing

Drop screenshots, annotated images, and play-by-play notes here.

## How to Use

1. Click around like a monkey
2. Screenshot anything that looks wrong or feels off
3. Save to `screenshots/` with a descriptive name
4. Add notes below or in a separate `take-N-notes.md`

## Template for Notes

```
## Take N -- [DATE]

### What I Tested
- [page/feature]

### What Worked
- ...

### What Broke (Houston We Have a Problem)
- Screenshot: screenshots/filename.png
- What happened: ...
- What I expected: ...

### What Felt Off (Not Broken, Just Weird)
- ...
```

## Bug Severity

- **BLOCKER** -- Can't record. Fix before take.
- **MAJOR** -- Visible on camera. Fix before take.
- **MINOR** -- Noticed it, won't show on camera. Fix after take.
- **COSMETIC** -- Pixel-pushing. Nice to have.
