# BorrowHood YouTube Playlist Plan

## Playlist Name

**"BorrowHood: The Garage Sessions"**

Why "Garage Sessions":
- The whole app is about garages becoming rental shops
- "Sessions" = walkthrough recordings, like jam sessions
- Covers happy path, edge cases, and "what went wrong" videos
- Sounds like a real series, not a one-off

Alternative names considered:
- "BorrowHood: From Garage to Garage" -- too literal
- "BorrowHood: Field Tests" -- good but clinical
- "BorrowHood: Break It, Fix It, Ship It" -- too long
- "BorrowHood: The Workshop Tapes" -- close second

---

## Video Plan (3 core + edge cases)

### Video 1: The Happy Path (DONE)
**Title:** BorrowHood: Full Rental Walkthrough -- From Browse to Completed
**What:** Complete rental flow. Sally borrows Mike's drill. Every step shown.
**Status:** Recorded, edited, YouTube kit ready.
**File:** `BorrowHood-Walkthrough-FINAL.mp4`

### Video 2: What Happens When Things Go Wrong
**Title:** BorrowHood: Disputes, Declines, and Cancellations
**What:** The other paths. What if Mike declines? What if Sally cancels? What if there's a dispute? Show:
- Mike declines a rental request (DECLINED path)
- Sally cancels a pending request (CANCELLED path)
- Sally reports a broken item (DISPUTED path)
- How the dispute resolution works
- Deposit held during dispute, released or forfeited on resolution
**Status:** Not started

### Video 3: The Community
**Title:** BorrowHood: Workshops, Help Board, and Your Neighborhood
**What:** Deep dive into the non-rental features:
- Creating a workshop profile from scratch
- Adding skills, languages, services
- Listing items (physical, digital, space, service, made-to-order)
- Help Board: posting a need, offering help, replies
- Badge system (Newcomer, Trusted, Pillar)
- Multilingual: English to Italian in one click
**Status:** Not started

### Edge Case Videos (future)

**"Broken When I Got It"**
- Sally picks up the drill and it's damaged
- She opens a dispute BEFORE using it
- Photos attached, message to Mike
- Mike sees the dispute, acknowledges
- Resolution: refund or replacement

**"I Broke It"**
- Sally returns the drill and it's damaged
- Mike opens a dispute AFTER return
- Deposit held, discussion via messages
- Resolution: partial deposit kept for repair

**"The Ghost Renter"**
- Sally never picks up (approved but no pickup)
- Auto-expiry after X days
- Mike's item goes back to available

**"The Lockbox Fail"**
- Wrong code entered
- Code expired
- What the renter sees, what the owner sees
