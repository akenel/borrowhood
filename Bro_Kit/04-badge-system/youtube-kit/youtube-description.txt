WHAT YOU'RE SEEING
- 15 achievement badges awarded for real milestones (first listing, first rental, 5-star review, etc.)
- 5 reputation tiers: Newcomer to Active to Trusted to Pillar to Legend
- Tier badges visible on every user card in the members directory
- Profile pages show earned badges with icons, colors, and descriptions
- Badge check API evaluates your activity and awards new badges automatically
- Points formula: items (10pts), rentals (20pts), giveaways (25pts), reviews (5pts), 5-stars (15pts)
- Language flags on profiles (GB, DE, FR, IT)
- OpenStreetMap location embed on profiles
- Share button and favorite hearts on items and profiles

BADGE LIST
First Listing, First Rental, First Review, Five Star, Super Lender, Trusted Renter, Community Voice, Multilingual, Skill Master, Auction Winner, Early Adopter, Perfect Record, Neighborhood Hero, Workshop Pro, Generous Neighbor

TIER THRESHOLDS
Newcomer (0pts) to Active (50pts) to Trusted (150pts) to Pillar (500pts) to Legend (1000pts)

TECH STACK
- FastAPI + SQLAlchemy (async)
- Alpine.js (badge rendering, favorites)
- Tailwind CSS
- Keycloak OIDC (session-based auth)
- PostgreSQL
- Docker Compose on Hetzner VPS
