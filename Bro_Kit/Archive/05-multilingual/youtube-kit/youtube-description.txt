Multilingual Live Switch -- One Click, Every Label, Instant | BorrowHood EP5

One click. The entire app switches language. Navigation, forms, badges, categories, filters, error messages -- everything. English to Italian and back, live on a production server.

We walk through 8 pages on a live Hetzner VPS. Each page loads in English, then switches to Italian with a single click. No page reload tricks. No partial translations. Full i18n coverage across the entire platform.

WHAT YOU ARE SEEING
- Full i18n with JSON locale files (en.json, it.json)
- Language switcher in the navigation bar with country flags
- URL-based language parameter (?lang=en, ?lang=it)
- Server-side rendering -- Jinja2 templates with t() translation function
- Every page translated: Home, Browse, Item Detail, Members, Help Board, Profile, List Item, Dashboard, Onboarding, Terms
- 500+ translated strings per language
- Categories (28), listing types (8), conditions (5), badge tiers (5), CEFR levels (7) -- all translated
- Rental status labels, auction terms, lockbox instructions, giveaway flow -- all in both languages
- Form placeholders, error messages, empty states -- nothing left in English-only
- Auto-translation for user-generated content (Alpine.js + browser API)

LANGUAGES
- English (en) -- full coverage
- Italian (it) -- full coverage
- Architecture supports adding more locales (drop in a JSON file)

TECH STACK
- FastAPI + Jinja2 (server-side i18n with t() helper)
- JSON locale files (structured, nested keys)
- Alpine.js (client-side dynamic content)
- Tailwind CSS
- Keycloak OIDC (session-based auth)
- PostgreSQL
- Docker Compose on Hetzner VPS
